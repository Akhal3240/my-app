# Simple Greeting App

## Description

This is a simple Android application built using **Kotlin** and **Jetpack Compose**.
The app displays a greeting message and a button. When the user clicks the button, the message changes.

## Features

* Displays a welcome message
* Button that changes the text when clicked
* Simple and clean user interface

## Technologies Used

* Kotlin
* Jetpack Compose
* Android Studio

## How to Run the App

1. Open the project in Android Studio.
2. Build the project.
3. Run the app on an Android emulator or a physical device.

## Purpose

This app was created to practice basic Android development concepts such as:

* Kotlin programming
* Compose UI components
* Button click events
